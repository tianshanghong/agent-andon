서랍 속에 잠자고 있는 그 오래된 iPad는 완벽한 **늘 켜져 있는 상태 보드**가 됩니다. 벽에 걸어 Agent Andon을 띄워 두면 모든 Claude Code와 Codex 에이전트가 한눈에 보입니다 — 완료되면 초록색, 손이 필요하면 주황색으로 — 그래서 확인하려고 굳이 alt-tab 할 일이 없습니다. 설치할 앱은 없습니다. 그냥 웹 페이지입니다.

## 보드 실행하기

에이전트가 돌아가는 머신에서:

```
andon serve
```

보드 URL이 출력됩니다. (아직 에이전트를 연결하지 않았다면? 먼저 `andon install claude` / `andon install codex`를 실행하세요.)

## iPad에서 열기

그 URL을 iPad의 **Safari**에서 여세요:

- **같은 Wi-Fi** — 출력된 LAN URL을 그대로 사용하세요.
- **어디서든** — Tailscale Serve로 보드를 노출하거나, 직접 운영하는 콘텐츠 블라인드 릴레이를 페어링한 뒤(`andon hosted setup <relay-url>`) 그 URL을 대신 여세요. [Hosted Andon](/ko/docs/hosted/)을 참고하세요.

그런 다음 **공유 → 홈 화면에 추가**로 브라우저 UI 없는 전체 화면을 만드세요.

## 항상 켜진 상태로 두기

iOS 설정 두 가지면 태블릿이 벽걸이 디스플레이로 바뀝니다:

- **설정 → 디스플레이 및 밝기 → 자동 잠금 → 안 함**, 그러면 화면이 꺼지지 않습니다.
- **가이드 접근**(설정 → 손쉬운 사용 → 가이드 접근)은 iPad를 보드에 고정해, 지나가다 화면을 한 번 눌러도 보드에서 벗어나지 않습니다.

## 거치하기

책상 위 저렴한 스탠드든, 눈높이의 벽걸이 거치대든 좋습니다. 이제 컨텍스트 전환이 아니라 곁눈질 한 번으로 어느 에이전트에 손이 필요한지 알 수 있습니다.

보드는 **손이 필요한** 세션을 맨 위로 띄우고 그 외에는 조용히 있습니다 — 그래서 iPad는 무언가 생기기 전까지는 차분합니다. 보드 서버에 대해서는 [Andon 실행하기](/ko/docs/running/)를, 데스크톱이나 휴대폰 알림도 원한다면 [알림](/ko/docs/notifications/)을 참고하세요.
